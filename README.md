## Description
The Hurricane Quiz Game is an interactive educational game designed to test players' knowledge about hurricanes, their formation, and their impact. Players can answer multiple-choice questions, use lifelines for assistance, and track their scores as they progress through different levels.

## Features
- **Multiple Levels:** The game includes questions across different difficulty levels.
- **Lifelines:** Players can use lifelines like 50/50 and Ask the Audience to help answer difficult questions.
- **Score Tracking:** The game keeps track of the player's score throughout the quiz.
- **Responsive Design:** The game is designed to work on both desktop and mobile devices.

## Technologies Used

## How to Play
1. Start the game by clicking on the "Start" button.
2. Read the question and select the correct answer from the available options.
3. Use lifelines if you're unsure about an answer.
4. Click "Next" to proceed to the next question.
5. At the end of the game, your total score will be displayed.
